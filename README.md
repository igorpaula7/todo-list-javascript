
# 📝 To Do List - JavaScript

Aplicação de lista de tarefas desenvolvida com **JavaScript puro**, permitindo adicionar, remover e concluir tarefas de forma dinâmica.

---

## 🚀 Funcionalidades

- ✅ Adicionar novas tarefas  
- ❌ Remover tarefas  
- ✔️ Marcar tarefas como concluídas  
- 🔽 Reordenar tarefas concluídas para o final da lista  

---

## 🛠 Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

---

## 💡 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de desenvolvimento front-end, como:

- Manipulação do DOM  
- Eventos (submit, click)  
- Criação dinâmica de elementos  
- Organização de código e estrutura  

---

## 🎯 Aprendizados

Durante o desenvolvimento deste projeto, foram aplicados conceitos como:

- Captura e tratamento de eventos do usuário  
- Uso de `createElement` para criação dinâmica de elementos  
- Utilização de `classList` para manipulação de estilos  
- Estruturação de lógica para interação com o usuário  

---

## 📂 Como executar o projeto

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/todo-list-javascript.git
```

2. Acesse a pasta do projeto:

```bash
cd todo-list-javascript
```

3. Abra o arquivo `index.html` no navegador
